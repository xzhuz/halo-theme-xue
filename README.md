## 支持的功能

- [x] 适配 Halo 最新 1.5.0 版本
- [x] 响应式主题
- [x] 代码高亮和代码行号
- [x] 可配置文章目录
- [x] 文章随机封面
- [x] 自定义网页标题
- [x] JavaScript 插件支持
- [x] 首页文章多样式支持
- [x] 全局 Pjax
- [x] 集成 fancybox 灯箱插件
- [x] 优化主题稳定性
